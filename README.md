## 📚 Topics Covered

### 🔢 NumPy
- Creating Arrays — 1D, 2D, 3D
- `zeros`, `ones`, `arange`, `linspace`, `eye`
- Array Indexing & Slicing
- Fancy Indexing & Boolean Masking
- Array Operations — add, subtract, multiply, divide
- Broadcasting
- Reshaping — `reshape`, `flatten`, `ravel`, `transpose`
- Mathematical Functions — `sqrt`, `exp`, `log`, `abs`
- Statistical Functions — `mean`, `median`, `std`, `var`, `min`, `max`
- Stacking & Splitting Arrays
- Random Module — `rand`, `randn`, `randint`, `seed`

### 🐼 Pandas
- Series & DataFrame — creation and basics
- Reading Data — `read_csv`, `read_excel`
- Exploring Data — `head`, `tail`, `info`, `describe`, `shape`
- Data Selection — `loc`, `iloc`, column selection
- Filtering & Conditional Selection
- Sorting — `sort_values`, `sort_index`
- Handling Missing Values — `isnull`, `dropna`, `fillna`
- Adding & Dropping Columns / Rows
- GroupBy & Aggregation
- Apply & Lambda functions
- Merge, Join & Concat
- Pivot Tables
- String Operations — `str` accessor
